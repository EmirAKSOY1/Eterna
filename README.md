
## Installation

```bash
git clone https://github.com/EmirAKSOY1/Eterna.git
```

## Usage

```bash
cd Eterna
cd blog
```
```bash
composer install
```
 .env copy
```bash
copy .env.example .env
```
# Generate Key
``` 
php artisan key:generate
```
# Npm run . I use Breeze
``` 
npm run dev
```
Then , run server
```
php artisan serve 
```

sql query : Eterna/eterna.sql
